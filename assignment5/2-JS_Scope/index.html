Javascript Scope

Javascript has 3 types of Scope

Block Scope

It is the scope which allows a variable to be used only in between {} curly braces. It can not be accessed outside {} block

Function Scope

It is the scope which allows a variable to be used in a function. It cannot be accessed outside that particular function

Global Scope

It is the scope which allows a variable to be used all over in the Javascript file.
